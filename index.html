The complete toolset developed for the TRACTION project

The TRACTION toolset is a collection of software applications for co-creation, dissemination, streaming, viewing and storing opera content.

  • Co-Creation Space is web-based platform which acts as a media repository and collaborative tool for upload, visualisation and communication around media objects.
    • The Face and Object Recognition Module for the Co-Creation Space contains neural networks capable of generating automatic tags based on face and object recognition for content filtering in the Co-Creation Space.
  • Co-Creation Stage is a web-based tool that supports both live and on-demand media to enable art professionals and individuals from the TRACT ION's targeted communities to remotely collaborate to produce engaging and immersive opera shows.
    • Pre Recorded Content Adaptation solution contains a DASH-based 2D web player and a novel audio-prioritised adaptation solution for 2D content streaming.
    • Live Content Adaptation solution contains a Web-RTC web player with adaptation of multiple real-time 2D video streams.
  • The Immersive Experience Engine of TRACTION contains two main tools: The Social VR component and the Immersive Adaptive Player.
    • Social VR component is an immersive environment for participants communications after attending a VR opera.
    • Immersive Adaptive Player is a web-based 360° video player with support to existing and novel adaptive DASH algorithms with audio prioritization, ambisonics support and accessibility features.


The toolset was originally developed as part of the TRACTION project, funded by the European Commission’s Horizon 2020 research and innovation programme under grant agreement No. 870610.
